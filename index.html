This page helps you see how csgo calculates floats

You can put in your desired float and it will show you the IEEE754 representation that csgo will output

Try adding or subtracting very small amounts to see what you should aim for

For example, 0.420 gives slightly less than 0.420 so try 0.420000005

It will then show you the minimum and maximum decimals that give the IEEE754 float you want

You can also see the exact middle which you may want to aim for just in case

This page is pretty simple

Enter in the min/max wear of the skin you want to make (found at https://csgostash.com/)

Along with what float you want and a couple other parameters

And enter in 10 or more floats and click generate

This will generate combinations of those 10 floats and show what float is the outcome (assuming it is within the range of how much to show over)

Make sure to double check any combinations with https://www.csgozone.net/. That way if the set of 10 skins are slightly off, you can sell them before failing a tradeup

This page is basically the same as the regular float combinations page, except you can load in data from opskins

You need to use your own api key for this

To get yours, go to: https://opskins.com/?loc=store_account#collapseAA

Input in the specifications of your search and click the "Add" button to add the data to the section below

To find minimum and maximum float values for a skin that you want to trade up to, check out https://csgostash.com/

You can add more than one search and select how many floats from each should be used to generate combinations

If you want one search to have more results than another, just change the parameters to be "x,x,x,x" to load x floats from each search (search0 - searchn)

I'm not trying to steal your api key, but because there are a lot of scammers and the code isn't currently open source, you can just enter in your specifications and the page will give you a link where you can put your own api key in

Once you have a tab open with the data, copy+paste it into the textarea below "Total Combinations Possible" (make sure there is a full line of space inbetween sets of data)

Make sure to double check any combinations with https://www.csgozone.net/. That way if the set of 10 skins are slightly off, you can sell them before failing a tradeup
